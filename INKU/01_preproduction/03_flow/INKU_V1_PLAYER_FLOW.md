# INKU V1 — Player Flow

## Main flow
Title → offline local entry → incubator → view current slime offer → buy if possible → auto-place in slot → inspect/manage slot → optional prairie presence → return incubator → close session → return later and receive offline income.

## Slot actions
- tap: focus/select
- drag: reorder
- long press: profile sheet

## Profile actions
- rename
- put in prairie / remove from prairie
- sell
- delete

## Prairie actions
- observe
- touch
- hold
- drag
- release
- return to incubator
