# INKU V1 — First Session Flow

## Recommended first-session experience
1. player launches game
2. enters offline local session
3. reaches incubator
4. sees slime, money, price, 4 slots
5. buys first slime
6. sees slot fill
7. opens profile
8. understands name, value, state
9. optionally renames
10. sends slime to prairie
11. sees living behavior
12. returns to incubator and starts the real loop

## First-session goal
Convert the player from “I see a creature” to “I own a specimen”.
