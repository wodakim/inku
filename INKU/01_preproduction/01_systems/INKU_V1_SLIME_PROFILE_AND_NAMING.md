# INKU V1 — Slime Profile and Naming

## Access
Open by long pressing a slot.

## Profile header
- large slime portrait
- displayed name
- rarity badge
- synthetic state

## Value section
- income/min
- resale value
- purchase price

## Dynamic state section
Shown in human-readable levels, not raw numbers:
- mood
- energy
- stress
- social need
- player trust

## Profile section
- 2 dominant displayed traits maximum
- 1 short descriptive sentence

## Actions
- Rename
- Put in prairie / Remove from prairie
- Sell
- Delete

## Naming rules
- default name exists for every slime
- custom name allowed
- max length: 16 characters
- trim spaces
- fallback to default name if empty

## Synthetic visible states
- Calm
- Curious
- Playful
- Tired
- Tense
- Nervous
- Unstable
- Sociable
