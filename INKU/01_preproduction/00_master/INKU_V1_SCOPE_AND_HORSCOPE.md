# INKU V1 — Scope and Horscope

## V1 includes

- Title screen entry
- Offline local session start
- Incubator scene with slime offer
- Money display
- Buying if enough money and slot available
- 4 persistent slots
- Auto-placement in first free slot
- Slot reorder
- Slime profile sheet
- Rename
- Sell
- Delete
- Passive income
- Offline income
- Prairie scene
- Up to 3 active slimes
- Minimal living behavior
- Minimal contextual co-presence
- Local persistence with save versioning

## Post-V1 but legitimate later

- persistent inter-slime affinity/fear/familiarity
- bounded personality evolution
- light conflict and recovery
- deeper player bond mechanics
- backend sync / cloud authority
- additional scenes and systems

## Explicitly out of scope now

- full combat
- deep social simulation
- relationship memory graph
- reproduction
- genetic inheritance depth
- inventory/crafting complexity
- heavy management-game economy
- production backend
- live-ops / marketplace / anti-cheat stack

## Drift alerts

Project is drifting if:
- too many stats appear
- slot UI becomes dense
- prairie becomes optimization-heavy
- economy gains multiple currencies or upgrade ladders
- deep social systems are added before V1 proof
- server architecture is pursued before local persistence is validated

## Filters for any new idea

A new idea enters V1 only if:
1. it directly improves the locked core loop
2. it is visible to the player
3. it preserves smartphone readability
4. it does not break scope discipline
